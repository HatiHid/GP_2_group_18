Вступление:














1) Цель проекта — не просто предсказать цену за м², а:

Показать как структура города (инфраструктура, транспорт, социальная среда) отражается в цене новостроек.

Определить какие факторы реально вносят вклад в стоимость

Найти зоны дисбаланса: где жильё стоит дорого, но инфраструктура слабая, где инфраструктура сильная, а цены ещё не успели вырасти.
